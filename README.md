## How to Use the Colab Notebooks

1. Click the **Open in Colab** badge next to any tutorial in the tables above.
2. In Colab, run the first setup cell — it installs all required libraries and loads the dataset automatically.
3. Work through the notebook from top to bottom. Cells marked **`# YOUR CODE HERE`** are gaps to fill in using what you learned in the lecture.
4. The interpretation questions in Section 4 ask you to reason about the output — write your answers as text. These cannot be answered by running more code.
5. Section 5 is an optional extension question for those who want to go further.
